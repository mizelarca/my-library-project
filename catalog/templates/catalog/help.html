{% if user_language == 'en' %}Heatmap{% else %}Тепловая карта{% endif %}

  • {% if user_language == 'en' %}Data is collected from two sources:{% else %}Данные собираются из двух источников:{% endif %}
    • {% if user_language == 'en' %}Reading timer — exact minutes per session (real time){% else %}Таймер чтения — точные минуты каждой сессии (реальное время){% endif %}
    • {% if user_language == 'en' %}Finished books — +60 minutes per book at the time it was added to your library{% else %}Прочитанные книги — +60 минут за каждую книгу в то время, когда она была добавлена в библиотеку{% endif %}
