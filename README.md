# Личная библиотека

Веб-приложение для учёта книг, создания коллекций и отслеживания прогресса чтения.

## Функции
- Добавление, редактирование, удаление книг
- Коллекции и теги
- Экспорт/импорт в CSV/XLSX
- Статистика и графики
- Тёмная тема и английский язык

## Технологии
- Python + Django
- MySQL
- Bootstrap 5
