# Личная библиотека

Веб-приложение для учёта книг, создания коллекций и отслеживания прогресса чтения.

## Основные возможности

### Управление книгами
- Добавление, редактирование и удаление книг
- Загрузка обложек и аватаров
- Рейтинг книг (оценка от 0 до 5 звёзд)
- Личные заметки и цитаты
- Дата начала и окончания чтения
- **Настроение книги** — выбор эмодзи для каждого прочитанного произведения (грустная, смешная, захватывающая, уютная и др.)
- **Музыка к книге** — добавление ссылок на треки и плейлисты из YouTube, Spotify, SoundCloud и других сервисов

### Таймер чтения
- Отслеживание реального времени чтения с точностью до секунды
- Большой дисплей с форматом ЧЧ:ММ:СС
- Кнопки Старт, Пауза, Стоп
- Ручное добавление времени (если забыли включить таймер)
- История последних сессий (за 30 дней)
- Удаление отдельных сессий
- Полный сброс времени по книге
- Подтверждение при закрытии страницы с активным таймером

### Тепловая карта чтения
- Визуализация активности по дням недели и часам суток (матрица 7×24)
- Данные из двух источников:
  - Реальные сессии таймера (точные минуты)
  - Прочитанные книги (+60 минут за каждую, по времени добавления в библиотеку)
- Цветовая шкала от светлого до тёмно-зелёного
- Статистика: количество сессий, прочитанных книг, общее время, пиковый час и день
- Адаптация под светлую и тёмную тему

### Коллекции
- Создание тематических коллекций
- Добавление книг в коллекции (по одной или массово)
- Ручной порядок книг внутри коллекции
- Массовое удаление коллекций

### Статистика и аналитика
- Круговая диаграмма статусов книг
- График прочитанных книг по месяцам
- Книжный вызов (цель на год + прогресс-бар)
- Топ-3 любимых авторов, жанров и лучших книг
- **Случайная цитата** на главной странице из ваших заметок

### Экспорт / Импорт
- Экспорт библиотеки в CSV (UTF-8 с BOM) и Excel XLSX
- Импорт книг из CSV/XLSX с автоматическим созданием авторов и жанров

### Интерфейс
- Адаптивная вёрстка (Bootstrap 5)
- **Тёмная и светлая тема** с переключением в профиле
- **Русский и английский язык** интерфейса
- Кнопка **«Что почитать?»** — случайная непрочитанная книга
- **Страница помощи** с подробным описанием всех функций

### Дополнительно
- Массовое удаление книг и коллекций
- Пагинация на всех страницах
- Поиск, фильтрация и сортировка книг
- Управление авторами и жанрами (список, добавление, удаление)
- Два режима просмотра книг: плитка и список (сохраняется выбор)
- Кликабельные карточки книг для быстрого перехода
