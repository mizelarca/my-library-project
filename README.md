# Личная библиотека

[![Django](https://img.shields.io/badge/Django-5.0-green)](https://www.djangoproject.com/)
[![MySQL](https://img.shields.io/badge/MySQL-8.0-blue)](https://www.mysql.com/)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-purple)](https://getbootstrap.com/)

Веб-приложение для учёта книг, создания коллекций и отслеживания прогресса чтения.

Основные возможности

### Управление книгами
- Добавление, редактирование и удаление книг
- Загрузка обложек и аватаров
- Рейтинг книг (оценка от 0 до 5 звёзд)
- Личные заметки и цитаты
- Дата начала и окончания чтения

### Коллекции
- Создание тематических коллекций
- Добавление книг в коллекции (по одной или массово)
- Ручной порядок книг внутри коллекции
- Массовое удаление коллекций

### Статистика и аналитика
- Круговая диаграмма статусов книг
- График прочитанных книг по месяцам
- Книжный вызов (цель на год + прогресс-бар)
- Топ-3 любимых авторов, жанров и лучших книг

### Экспорт / Импорт
- Экспорт библиотеки в CSV (UTF-8 с BOM) и Excel XLSX
- Импорт книг из CSV/XLSX с автоматическим созданием авторов и жанров

### Интерфейс
- Адаптивная вёрстка (Bootstrap 5)
- Тёмная и светлая тема
- Русский и английский язык интерфейса
- Кнопка «Что почитать?» — случайная непрочитанная книга

### Дополнительно
- Массовое удаление книг и коллекций
- Пагинация на всех страницах
- Поиск, фильтрация и сортировка книг
- Управление авторами и жанрами (список, добавление, удаление)

## Технологии

| Компонент | Технология |
|-----------|------------|
| Backend | Python 3.10 + Django 5.0 |
| База данных | MySQL 8.0 |
| Frontend | HTML5, CSS3, Bootstrap 5 |
| Графики | Chart.js |
| Экспорт/импорт | django-import-export, openpyxl |
| Сервер | Ubuntu 22.04, Gunicorn, Nginx |
